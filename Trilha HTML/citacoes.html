O Elemento HTML <blockquote> (ou Elemento HTML de citação de bloco ) indica que o texto incluído é uma longa citação.

De acordo com página MDN blockquote

O elemento quote - <q> - é usado quando otexto dentro da tag é uma pequena citação. Este elemento destina-se a citações curtas que não requerem marcações de parágrafo; para citações maiores use o elemento <blockquote> . -- MDN q page
